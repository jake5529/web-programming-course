# LR11 Standalone Slides

Standalone версия слайдов LR11 (DevOps - Docker & CI/CD, local-first) со встроенными локальными зависимостями.

## Использование

1. Откройте файл `slides.html` в браузере
2. Навигация: стрелки или пробел
3. `ESC` - overview режим
4. `S` - speaker notes

## Что входит

- Reveal.js runtime и плагины
- Theme CSS + syntax highlight
- Полностью offline-режим

## Горячие клавиши

- `←` / `→` - переход между слайдами
- `Space` - следующий слайд
- `F` - fullscreen
- `ESC` - обзор всех слайдов
- `B` - black screen
